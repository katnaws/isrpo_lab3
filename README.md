# CATastrophic errros - плагин для отображения мемов с котиками по количеству ошибок

Плагин автоматически отслеживает:
- количество ошибок в открытом файле,

- переключение вкладок,

- изменения диагностик компилятора/языкового сервера.

И обновляет картинки в отдельной панели.

# Что нужно сделать, чтобы скачать этот плагин себе?
1. Склонировать репозиторий к себе:  
``` git clone https://github.com/katnaws/isrpo_lab3.git```  
2. Открыть проект в VS Code  
3. Запустить debug в файле ```extension.js```  
4. Откроется новое окно среды разработки, теперь можно открывать в нем свой код и редактировать его вместе с котиками!
